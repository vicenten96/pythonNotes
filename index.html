Operadores Aritméticos

Python obedece las reglas matemáticas de la precedencia de Operadores

  • SUMA: +
  • RESTA: -
  • MULTIPLICACION: *
  • DIVIDIR: /
  • DIVISION ENTERA: //
  • MODULO: %
  • POTENCIA: **

Variables

Las variables en python son como cajas que contiene objetos, estas puenden guardar uno o varios objetos a la vez.

El nombre que se le asigna a la variable se le conoce como IDENTIFICADOR.

numero = 3

Siendo:

-"numero" el identificador
-"=" asignador
-"3" el valor de numero

Datos

  • Numeros Enteros(int). ej:4
  • Numero de punto flotante(float). ej:4.5
  • Texto(str). ej:"Hola Mundo"
  • Booleanos(True or False). ej:True

Se pueden sumar textos, a esto se le llama CONCATENACIÓN. Tambien multiplicar variables que contienen cadenas de texto.

Operadores lógicos

  • Operador "and": Python va a devolver verdadero cuando todas las variables comparadas contengan el valor de True.

    ej:

    >_True or False
    >>>False

  • Operador "or": Python va a devolver verdadero siempre que una de las variables comparadas sea verdadera.

    ej:

    >_True or False
    >>>True

  • Operador "not": El operador not lo que va a hacer es invertir el valor de una variable.

    ej:

    >_not True
    >>>False

Operadores de comparación

  • Operador "igual a"( == ): El operador igual a funciona para conocer si dos variables tienen el mismo valor.

    ej:

    >_5 == 5
    >>>True

  • Operador "distinto a"( != ): El operador distinto a funciona para conocer si dos variables son distintas entre ellas. Es el opuesto del operador "igual a"

    ej:

    >_5 != 5
    >>>False

  • Operador "mayor que y menor que"( > y < ): El operador mayor que funciona para conocer una variables es mayor que la segunda variable y el operador menor que funciona de manera inversa.

    ej:

    >_5 > 4
    >>>True

    >_5 < 4
    >>>False

  • Operador "mayor igual que y menor igual que"( > y < ): El operador mayor igual que funciona para conocer una variables es mayor o igual que la segunda variable y el operador menor igual que funciona de manera inversa.

    ej:

    >_5 >= 5
    >>>True

    >_5 >= 8
    >>>False

    >_5 <= 2
    >>>False
