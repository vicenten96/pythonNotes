Operadores Aritmeticos

Python obedece las reglas matematicas de la precedencia de Operadores

  • SUMA: +
  • RESTA: -
  • MULTIPLICACION: *
  • DIVIDIR: /
  • DIVISION ENTERA: //
  • MODULO: %
  • POTENCIA: **

Variables

Las variables en python son como cajas que contiene objetos, estas puenden guardar uno o varios objetos a la

El nombre que se le asigna a la variable se le conoce como IDENTIFICADOR.

numero = 3

Siendo:

-"numero" el identificador
-"=" asignador
-"3" el valor de numero

Datos

  • Numeros Enteros(int). ej:4
  • Numero de punto flotante(float). ej:4.5
  • Texto(str). ej:"Hola Mundo"
  • Booleanos(True or False). ej:True

Se pueden sumar textos, a esto se le llama CONCATENACION. Tambien multiplicar variables que contienen cadenas de texto.
